## 🎨 Цветовые коды

### Основные:
- **Бренд зеленый**: `#7a9375` → `#a4b8a0`
- **Золотой**: `#ffd700` → `#ffed4e`
- **Белый фон**: `#ffffff` / `rgba(255,255,255,0.9)`
- **Текст**: `#1d1d1f`

### Подарки:
1. **Pink**: `#ff6b9d` → `#ffb3c1`
2. **Gold**: `#ffd700` → `#fff9a5`
3. **Green**: `#7a9375` → `#c5d4c1`
4. **Purple**: `#9b87f5` → `#d0c4fa`
5. **Orange**: `#f97316` → `#fdba74`
6. **Cyan**: `#06b6d4` → `#67e8f9`

### Фон:
- **Верх**: `#e8f4e8`
- **Середина**: `#f5f9f5`
- **Низ**: `#fff9f0`

## 🎯 Интерактивные эффекты

### Hover (при наведении):
```
Обычный:     Hover:
┌────┐      ┌────┐
│ 🎁 │  →   │ 🎁 │  (увеличивается 1.15x)
└────┘      └────┘  (поворот ±5°)
                    (зеленое кольцо)
```

### Tap (при клике):
```
Нажатие:
┌────┐
│ 🎁 │  (уменьшается 0.9x)
└────┘  (быстро)
```

### После выполнения:
```
   ✨ → ⭐ → 💫 → 🌟   (по кругу)
          ↓
        🎁🎀
        (пульсирует)
```
