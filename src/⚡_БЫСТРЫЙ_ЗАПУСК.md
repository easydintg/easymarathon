### ШАГ 3: Тест (2 минуты)

1. Открой бота в Telegram
2. Нажми кнопку меню **"🎯 Начать марафон"**
3. Проверь:
   - ✓ Видны подарки в зигзаге
   - ✓ Надписи "День 1", "День 2"
   - ✓ Елочки между подарками
   - ✓ Можно кликнуть на День 1
   - ✓ Можно выполнить задание
   - ✓ Прогресс сохраняется

## 🐛 Быстрые фиксы

### "Белый экран"
→ Проверь Environment Variables в Vercel
→ Проверь что index.html с Telegram SDK

### "Не открывается"
→ Проверь URL в /setmenubutton (должен быть с https://)
→ URL без слэша в конце

### "Не сохраняется"
→ Это норм! Работает через localStorage
→ Закрой и открой бота - прогресс на месте?
